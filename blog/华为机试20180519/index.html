简单密码

手机上的字母： 1–1， abc–2, def–3, ghi–4, jkl–5, mno–6, pqrs–7, tuv–8 wxyz–9, 0–0,

而密码中出现的大写字母则变成小写之后往后移一位，如：X，先变成小写，再往后移一位，就是y了
z往后移是a
例：输入：YUANzhi1987
输出：zvbo9441987

    public static void password(String s) {
    char[] ch =s.toCharArray();
    for(int i=0;i<s.length();i++){
        if(s.valueOf(ch[i]).matches("[A-Y]")){
            ch[i]=(char)(s.toLowerCase().charAt(i)+1);

        }else if(ch[i]=='Z'){
            ch[i]='a';
        }else if(s.valueOf(ch[i]).matches("[abc]")){
            ch[i]='2';
        }else if(s.valueOf(ch[i]).matches("[def]")){
            ch[i]='3';
        }else if(s.valueOf(ch[i]).matches("[ghi]")){
            ch[i]='4';
        }else if(s.valueOf(ch[i]).matches("[jkl]")){
            ch[i]='5';
        }else if(s.valueOf(ch[i]).matches("[mno]")){
            ch[i]='6';
        }else if(s.valueOf(ch[i]).matches("[pqrs]")){
            ch[i]='7';
        }else if(s.valueOf(ch[i]).matches("[tuv]")){
            ch[i]='8';
        }else if(s.valueOf(ch[i]).matches("[wxyz]")){
            ch[i]='9';
        }
    }
    for(char c:ch){
        System.out.print(c);
    }

}

删除字符串出现最少的字符

实现删除字符串中出现次数最少的字符，若多个字符出现次数一样，则都删除。输出删除这些单词后的字符串，字符串中其它字符保持原来的顺序。

    public static void main(String[] args) {
    Scanner cin = new Scanner(System.in);
    while(cin.hasNext()) {
    int[] num = new int[26];//分配为大小26的数组，初始默认为0
    String line = cin.nextLine();
    for(int i=0; i<line.length(); i++) {
    char c = line.charAt(i);
    int n = (int)(c - 'a');//这里减a等于把所有字符都变为0-26的数字，a=0
    num[n] ++;//将字符的个数一一加入对应的数组位置
    }
    int min = Integer.MAX_VALUE;
    for(int i=0; i<26; i++) {
    if(num[i] != 0 && num[i] < min) //判断出最小的字符的数量
    min = num[i];
         }
     }
    for(int i=0; i<line.length(); i++) 
      char c = line.charAt(i);
         int n = (int)(c - 'a');
        if(num[n] != min) {//每次判断这个字符是否属于最小数量的字符，如果不是则输出。
         System.out.print(c);
         }
     }
        System.out.println();
    }
}

输出质数因子

功能:输入一个正整数，按照从小到大的顺序输出它的所有质数的因子（如180的质数因子为2 2 3 3 5 ）

  public static void main(String[] args){
    Scanner in=new Scanner(System.in);
    long x=in.nextLong();
    getResult(x);
}
public static void getResult(long x){
    long y=2;
    while(y<=x){//每次都除2，直到没有2的因子为止，然后三，等于自动就排序了
        if(x%y==0){
            System.out.print(y+" ");
            x/=y;
        }
        else{
            y++;
        }
    }
}

合并数据表

数据表记录包含表索引和数值，请对表索引相同的记录进行合并，即将相同索引的数值进行求和运算，输出按照key值升序进行输出。

public static void main(String[] args){
        Scanner in=new Scanner(System.in);
        int x=in.nextInt();
        Map<Integer,Integer> p=new TreeMap();
        for(int y=0;y<x;y++){
            int a =in.nextInt();
            int b=in.nextInt();
            if(p.containsKey(a)){//判断键是否有重复
                int c=(int)p.get(a)+b;//值相加
                p.put(a, c);
            }else{
                p.put(a, b);
            }
        }
        for(Map.Entry<Integer,Integer> m : p.entrySet() ){
            System.out.println(m.getKey()+" "+m.getValue());
        }
    }

提取不重复的正数

输入一个int型整数，按照从右向左的阅读顺序，返回一个不含重复数字的新的整数。

public static void main(String[] args) {
    Scanner in= new Scanner(System.in);
    String str = in.nextLine();
    char[] b=str.toCharArray();
    //System.out.println(Arrays.toString(b));
    LinkedHashSet hs=new LinkedHashSet<Integer>();//数组set的去重
    for(int a=0;a<b.length;a++){
        hs.add(b[b.length-1-a]);
    }
    Object[] arr=hs.toArray();
    for(Object a:arr){
        System.out.print(a);
    }
}

字符个数统计

编写一个函数，计算字符串中含有的不同字符的个数。字符在ACSII码范围内(0~127)。不在范围内的不作统计

public static void main(String[] args) {
    Scanner in = new Scanner(System.in);
    char[] ch = in.nextLine().toCharArray();
    int count=0;
    char[] ch1=new char[127];
    for(int x=0;x<ch.length;x++){
        int m=(int)ch[x];

        if(m>=0&&m<=127&&ch1[m]!=1){//这里ch1[m]！=1是为了判断这个之前有没有出现过，没有出现过，个数加1
            count++;
        }
        ch1[m]=1;
    }
    System.out.println(count);
}

坐标移动

开发一个坐标计算工具， A表示向左移动，D表示向右移动，W表示向上移动，S表示向下移动。从（0,0）点开始移动，从输入字符串里面读取一些坐标，并将最终输入结果输出到输出文件里面。

合法坐标：A10

非法坐标点需要进行丢弃。如AA10; A1A; $%$; YAD; 等。

public static void main(String[] args) {
    Scanner in = new Scanner(System.in);
    while(in.hasNextLine()){
    int x=0,y=0;

    String[] s=in.nextLine().split(";");
    for(int a=0;a<s.length;a++){
        int b=Integer.parseInt(s[a].substring(1));
        try {
            if(s[a].charAt(0)=='A'){
                x-=b;
            }
             if(s[a].charAt(0)=='W'){
                y+=b;
            }
             if(s[a].charAt(0)=='S'){
                y-=b;
            }
             if(s[a].charAt(0)=='D'){
                x+=b;
            }
        } catch (NumberFormatException e) {
            continue;
        }
    }
    System.out.println(x+","+y);
}
}

密码验证合格程序

密码要求:
1.长度超过8位
2.包括大小写字母.数字.其它符号,以上四种至少三种
3.不能有相同长度超2的子串重复
说明:长度超过2的子串
输入描述:
一组或多组长度超过2的子符串。每组占一行
输出描述:
如果符合要求输出：OK，否则输出NG

public static void main(String[] args) {
    // TODO Auto-generated method stub
    Scanner in = new Scanner(System.in);
    while(in.hasNextLine()){
    String s = in.nextLine();
    password(s);
    }
}

public static void password(String s) {
    // TODO Auto-generated method stub
    int count=0,up=0,down=0,num=0;
    if(s.length()<9){
        System.out.println("NG");
        return;
    }
    for(int i=0;i<s.length();i++){

        if(s.valueOf(s.charAt(i)).matches("\\d")){
            count=1;
        }else if(s.valueOf(s.charAt(i)).matches("[A-Z]")){
            up=1;
        }else if(s.valueOf(s.charAt(i)).matches("[a-z]")){
            down=1;
        }else{
            num=1;
        }
    }
    if(f(s)==true&&count+up+down+num>=3){
        System.out.println("OK");
    }else{
        System.out.println("NG");
    }
}
public static boolean f(String s){//3.不能有相同长度超2的子串重复
    for(int i=0;i<s.length()-3;i++){
        if(s.substring(i+1).contains(s.substring(i, i+3))){
            return false;
        }
}
    return true;
}

位运算

求一个值的二进制中1的个数

n&n-1的妙用：

  n&(n-1)作用：将n的二进制表示中的最低位为1的改为0，先看一个简单的例子：

 n = 10100(二进制），则(n-1) = 10011 ==》n&(n-1) = 10000

 可以看到原本最低位为1的那位变为0。



方法一：
public class Main{
    public static void main(String[] args){
        Scanner scan = new Scanner(System.in);
        int n = scan.nextInt();
        int count = 0;
        while(n>0){
            if((n&1)>0){
                count++;
            }
            n=n>>1;
        }
        System.out.println(count);
    }
}

方法2：
public class Main {
public static void main(String[] args) {
    Scanner sc=new Scanner(System.in);
    while (sc.hasNext()){
        int n=sc.nextInt();
        int count=0;
        while (n!=0){//只要n不等于0，必定还有1，所以count+1
            count++;
            n=n&(n-1);
        }
        System.out.println(count);
    }
}

}
