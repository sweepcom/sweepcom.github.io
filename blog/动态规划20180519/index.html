动态规划

发表于 2018-05-19 | 分类于 算法

动态规划定义

动态规划过程是：每次决策依赖于当前状态，又随即引起状态的转移。一个决策序列就是在变化的状态中产生出来的，所以，这种多阶段最优化决策解决问题的过程就称为动态规划

动态规划实际上是一类题目的总称，并不是指某个固定的算法。动态规划的意义就是通过采用递推（或者分而治之）的策略，通过解决大问题的子问题从而解决整体的做法。动态规划的核心思想是巧妙的将问题拆分成多个子问题，通过计算子问题而得到整体问题的解。而子问题又可以拆分成更多的子问题，从而用类似递推迭代的方法解决要求的问题。

动态规划的应用场景

对于一个可拆分问题中存在可以由前若干项计算当前项的问题可以由动态规划计算。
能用动态规划计算的问题存在一种递推的连带关系。

分析问题时需要进行状态转换，分解问题，分解成自己能够解决的问题

例：题目：求一个数列中最大连续子序列的和。

原问题转化为：Fk是第k项前的最大序列和，求F1～FN中最大值。

#动态规划的应用举例 #

数塔取数问题

一个高度为N的由正整数组成的三角形，从上走到下，求经过的数字和的最大值。
每次只能走到下一层相邻的数上，例如从第3层的6向下走，只能走到第4层的2或9上。

该三角形第n层有n个数字，例如：

第一层有一个数字：       5

第二层有两个数字：      8 4

第三层有三个数字：     3 6 9

第四层有四个数字：    7 2 9 5

最优方案是：5 + 8 + 6 + 9 = 28

状态定义: Fi，j是第i行j列项最大取数和，求第n行Fn，m（0 < m < n）中最大值。

状态转移方程：Fi，j = max{Fi-1,j-1,Fi-1,j}+Ai,j

上述解释可能还明白，看代码分析：

public static void main(String[] args) {
   Scanner in = new Scanner(System.in);
   int n = in.nextInt();
   long max = 0;//用来获取最大路径
   int[][] arr=new int[n][n];     //建立数组存放值
   arr[0][0]=in.nextInt();        //将第一行值存入
   for(int i=1;i<n;i++){          //共几行，就循环几次
       for(int j=0;j<=i;j++){
           int num=in.nextInt();
           if(j==0){
               arr[i][j]=arr[i-1][j]+num;//每行第一个值是上一行 第一个值加上给出的值
           }else{
               arr[i][j]=Math.max(arr[i-1][j-1], arr[i-1][j])+num;
                //每行后面的值都取上一行相邻俩个值大的再加上新值
           }
           max=Math.max(arr[i][j], max);//每次都与max比较，时刻保持最大路径
       }
   }
   System.out.println(max);
}    
}

解析：其实动态规划就是分解成多个子问题，问题是求最大路径，那么就把每一行的最大路径存进去
     ，每次加入新的一行只需要获取前一行的值就可以了，不用在重新重头走一遍了
如下

                    三角形数字                    动态规划过程
第一层有一个数字：       5                             5

第二层有两个数字：      8 4                        13  9

第三层有三个数字：     3 6 9                     16  19 18

第四层有四个数字：    7 2 9 5                  23  21  28 23

第一行直接存入5
第二行存的是最路径的和（8+5）（4+5）
第三行这里计算的时候，如（19）只需要原值6加上上一行相邻值的较大者13，得到19
不需要在重新进行5+8+6的计算了
所以每次得到的都是在那个点的最大路径值，分而治之。

矩阵取数问题

一个NN矩阵中有不同的正整数，经过这个格子，就能获得相应价值的奖励，从左上走到右下，只能向下向右走，求能够获得的最大价值。例如：3 3的方格。

1 3 3 
2 1 3 
2 2 1

能够获得的最大价值为：11。

解题思路同上
public static void main(String[] args) {
    Scanner in = new Scanner(System.in);
    int n = in.nextInt();
    int[][] arr = new int[n+1][n+1];//这里用n+1是为了下面的i-1=0时的角标存在，
                                        不需要重新单独考虑，默认从角标1开始
    int max=Integer.MIN_VALUE;
    for(int i=1;i<n+1;i++){
        for(int j=1;j<n+1;j++){
            int num =in.nextInt();
            if(i==1){
                arr[i][j]=arr[i][j-1]+num;
            }
            else{
                arr[i][j]=Math.max(arr[i][j-1], arr[i-1][j])+num;
            }
            max=Math.max(arr[i][j], max);
        }
    }
    System.out.println(max);
}

最长递增子序列

给出长度为N的数组，找出这个数组的最长递增子序列。 
(递增子序列是指，子序列的元素是递增的） 
例如：5 1 6 8 2 4 5 10，最长递增子序列是1 2 4 5 10。

O(n^2)
public static void main(String[] args) {
     Scanner in = new Scanner(System.in);
        int n = in.nextInt();
        int[] A = new int[n];
        for (int j = 0; j < n; j++){
            A[j] = in.nextInt();
        }
        int[] dp=new int[n];//用来存放每个数字的最长子序列长度
        int max=0;
        for(int i=0;i<n;i++){
            dp[i]=1;//每个数的初始序列长度是自身也就是1.
            for(int j=0;j<i;j++){
                if(A[i]>A[j]){
                    dp[i]=Math.max(dp[i],dp[j]+1);//让比这个数小的所有值的子序列相比
                                                    取最大值然后加1；
                    max=Math.max(dp[i]，max）;
            }     
        }
        }
        System.out.println(max);
}

解析：A里面用来存放原始数组
     dp里面用来存放以各自原始数值为末尾时的最小子串长度
     这种方法时间复杂度O（n^2），类似与冒泡了，每次都要与之前的所有相比
     例如8，上述程序要与5的最长子序列也就是1相比，要和1的子序列长度1比，要和6的子序列长
     度2相比，但是其实只需要和6比了，在时间复杂度上做的不够，但是代码更加简洁

O(nlogn)

public int findLongest(int[] A, int n) {
        int[] B = new int[n+1]; B[1] = A[0];
        int len=1,start=0,end=len,mid;
        for(int i = 1;i<n;i++){
        if(A[i]>B[len]) {
            len++;
            B[len] = A[i];
        }
        else{
            start=1;end=len;
            while(start<=end){
            mid=(start+end)/2;
            if(B[mid]<A[i]) start=mid+1;
            else end=mid-1;
            } 
            B[start] = A[i];
        }
    }
    }
    return len;

求最大子段和

例如：-2,11,-4,13,-5,-2，和最大的子段为：11,-4,13。和为20。

public static void main(String[] args) {
     Scanner in = new Scanner(System.in);
        int n = in.nextInt();
        long max=0;long summax=0;
        for(int i=0;i<n;i++){
            summax+=in.nextLong();
            if(summax>max){//当最新字段大于之前字段时，更新max
                max=summax;
            }
            if(summax<0){当子段小于0时，重头开始计算
                summax=0;
            }
        }
        System.out.println(max);
}
