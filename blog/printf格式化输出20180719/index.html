发表于 2018-07-19 | 分类于 Java基础

printf支持的格式

目前printf支持以下格式： 
      %c        单个字符 
      %d        十进制整数 
      %f        十进制浮点数 
      %o        八进制数 
      %s        字符串     %S全部大写输出
      %u        无符号十进制数 
      %x        十六进制数 
      %%        输出百分号% 
      %n        表示换行
      %t        表示时间

①%：表示格式说明的起始符号，不可缺少。 
②-：有-表示左对齐输出，如省略表示右对齐输出。 
③0：有0表示指定空位填0,如省略表示指定空位不填。 
④m.n：m指域宽，即对应的输出项在输出设备上所占的字符数。N指精度。用于说明输出的实型数的
 小数位数。为指定n时，隐含的精度为n=6位。 
⑤l或h:l对整型指long型，对实型指double型。h用于将整型的格式字符修正为short型。


double d = 345.678;
String s = "hello!"; 
int i = 1234;

输出浮点数 %f

1.System.out.printf("%f",d);
  输出：345.678000，//"f"表示格式化输出浮点数。

2.System.out.printf("%.2f",d);//
  输出：345.68//没有限制长度，小数后留俩位
3.System.out.printf("%9.2f",d);//"9.2"中的9表示输出的长度，2表示小数点后的位数。
  输出：‘    45.68’ 9表示长度为9，长度不够左边补空格，2表示小数点后俩位

4.System.out.printf("%+9.2f",d);//"+"表示输出的数带正负号。

5.System.out.printf("%-9.4f",d);//"-"表示输出的数左对齐（默认为右对齐，如例2）。
  输出：“45.6780 ”  小数后不够则补0；长度不够补空格

6.System.out.printf("%+-9.3f",d);//"+-"表示输出的数带正负号且左对齐。

7.System.out.printf("%+-9.3f%n",d)输出完换行

输出整数

1.System.out.printf("%d",i);"d"表示输出十进制整数。
  输出：1234
2.System.out.printf("%o",i);//"o"表示输出八进制整数。

3.System.out.printf("%x",i);//"d"表示输出十六进制整数。
  输出：4d2
4.System.out.printf("%#x",i);//"d"表示输出带有十六进制标志的整数。
  输出：0x4d2

5.System.out.printf("%s",s);//"d"表示输出字符串

输出日期

日期的俩种常用组合：

System.out.printf("%1$tD%n",date);
System.out.printf("%1$tF%n",date);1$这里可以省略，因为只有一个值

输出：07/19/18
     2018-07-19

时间常见的组合：
System.out.printf("%1$tR%n", date);
System.out.printf("%1$tT%n", date);
System.out.printf("%1$tr%n", date);
输出：
    15:04
    15:04:33
    03:04:33 下午

输出日期完整信息：
System.out.printf("%1$tc%n", date);
System.out.println(date);
输出：  

    星期四 七月 19 15:06:40 CST 2018
    Thu Jul 19 15:06:40 CST 2018

输出百分比

double d = 0.67847; 

1.System.out.printf("%.2f%%",d*100 );
  输出：67.85%

2.System.out.printf("%.2f%%",d);
  输出：0.68%
