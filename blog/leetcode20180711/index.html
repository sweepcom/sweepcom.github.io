1.求俩个数的和

给定一个整数数组，找出其中两个数满足相加等于你指定的目标数字。

　　要求：这个函数twoSum必须要返回能够相加等于目标数字的两个数的索引，且index1必须要小于index2。请注意一点，你返回的结果（包括index1和index2）都不是基于0开始的。你可以假设每一个输入肯定只有一个结果。

暴力遍历：
public int[] twoSum(int[] nums, int target) {
    int[] arr=new int[2];
    for(int i=0;i<nums.length;i++){
        for(int j=i+1;j<nums.length;i++){
            if(nums[i]+nums[j]==target){
                arr[0]=i;
                arr[1]=j;
                return arr;
            }
        }
    }
    return arr;
}

两遍哈希表

2.俩数相加

给定两个非空链表来表示两个非负整数。位数按照逆序方式存储，它们的每个节点只存储单个数字。将两数相加返回一个新的链表。

输入：(2 -> 4 -> 3) + (5 -> 6 -> 4)
输出：7 -> 0 -> 8
原因：342 + 465 = 807

你可以假设除了数字 0 之外，这两个数字都不会以零开头。

 public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
    int a=0,b=0,n1=0,n2=0;
    while(l1!=null){
        a+=l1.val*Math.pow(10,n1);
        n1++;
        l1=l1.next;
    }
    while(l2!=null){
        b+=l2.val*Math.pow(10,n2);
        n2++;
        l2=l2.next;
    }
    int c=a+b;
    ListNode head=new ListNode(0);
    ListNode result=head;
    while(c!=0){
        int temp=c%10;
        c=c/10;
        head.next=new ListNode(temp);
        head=head.next;
    }
    return result.next;
}

3.无重复的最长子串

给定 “abcabcbb” ，没有重复字符的最长子串是 “abc” ，那么长度就是3。

 public int lengthOfLongestSubstring(String s) {
    int len=0;
    Map<Character,Integer> m=new HashMap();  //
    for(int i=0,j=0;j<s.length();j++){ 
            //i代表子串重新开始的计算位置
        if(m.containsKey(s.charAt(j))){
            i=Math.max(m.get(s.charAt(j)),i);
        }
        len=Math.max(len,j-i+1);//
        m.put(s.charAt(j),j+1);//当前字符及其在字符串中的下一个位置
    }
    return len;
}

如abcadjh：
m一开始存的a，1，后面在遇到第二个a的时候，将i的值设为之前a和i较大的值1，也就是从bca开
始继续,没遇到重复字符都这么计算，直到遍历完字符串

4.俩个排序数组的中位数

nums1 = [1, 3]
nums2 = [2]

中位数是 2.0

nums1 = [1, 2]
nums2 = [3, 4]

中位数是 (2 + 3)/2 = 2.5

 public double findMedianSortedArrays(int[] nums1, int[] nums2) {
    int n1=nums1.length;
    int n2=nums2.length;
    int[] arr=new int[n1+n2];
    int k=0,i=0,j=0;

    while(i<n1&&j<n2){
        if(nums1[i]<nums2[j])
            arr[k++]=nums1[i++];
        else
            arr[k++]=nums2[j++];
    }
    while(i<n1){
        arr[k++]=nums1[i++];
    }
    while(j<n2){
        arr[k++]=nums2[j++];
    }
    int mid=arr.length/2;
    if(arr.length%2==1)
        return (double)(arr[mid]);

    return ((double)(arr[mid]+arr[mid-1])/2);

#5.最长回文子串 #
动态规划，每次遍历都找到以单个字符和以双个字符为中心的最长回文串

public String longestPalindrome(String s) {
int start = 0, end = 0;
for (int i = 0; i < s.length(); i++) {
    int len1 = expandAroundCenter(s, i, i);//判断以该字符为中心的回文长度
    int len2 = expandAroundCenter(s, i, i + 1);//以该字符及下一个为中心
    int len = Math.max(len1, len2);
    if (len > end - start) {
        start = i - (len - 1) / 2;
        end = i + len / 2;
    }
}
return s.substring(start, end + 1);
}
//返回回文串长度
private int expandAroundCenter(String s, int left, int right) {
    int L = left, R = right;
    while (L >= 0 && R < s.length() && s.charAt(L) == s.charAt(R)) {
        L--;
        R++;
    }
    return R - L - 1;//这里最后left和right都不是最终角标，
                        如aba最后，left=-1；right=3，所以要计算
}

6.俩数相减

输入一个数组，由数字构成，可以表示一个数，反转该数组得到B，求A与B的差的绝对值

如[1,2,3],321-123=198,输出[1，9，8]
121，返回[0]

public static int[] solve(int[] A){
    int[] B=new int[A.length];
    int temp=0;//用来表示进位
    if(judge(A)==true){ //正序大于逆序
        for(int i=0,j=A.length-1;i<A.length;i++,j--){
            if(A[i]-temp>=A[j]){  
                B[j]=A[i]-A[j]-temp; 不需要进位的情况
                temp=0;
            }else{
                B[j]=A[i]-A[j]-temp+10; //需要进位的情况
                temp=1;
            }
        }
    }
    if(judge(A)==false){ //逆序大于正序
        for(int i=0,j=A.length-1;i<A.length;i++,j--){
            if(A[j]-temp>=A[i]){
                B[j]=A[j]-A[i]-temp;
                temp=0;
            }else{
                B[j]=A[j]-A[i]-temp+10;
                temp=1;
            }
        }
    }
    int index=0;
    while(index<B.length&&B[index]==0){
        index++;
    }
    int[] C=new int[B.length-index];
    for(int i=0;i<C.length;i++){
        C[i]=B[index++];
    }
    if(C.length==0)
        return new int[]{0};
    return C;
}
public static boolean judge(int[] arr){ //判断数组和数组翻转后谁大决定谁减谁
    int i=0,j=arr.length-1;
    while(i<arr.length){
        if(arr[i]<arr[j]){
            return true;
        }
        else if(arr[i]>arr[j]){
            return false;
        }
        else{
            i++;
            j--;
        }
    }
    return true;
}

7.全排列

每次固定一个位置，如1，2，3，固定1，交换后面的所有，固定2，交换其他的
给定一个没有重复数字的序列，返回其所有可能的全排列。
如果是有重复数字的可以set来删除

public List<List<Integer>> permute(int[] nums) {
      List<List<Integer>> result=new ArrayList<>();
       permutation(nums,0,result);
       return result;
   }
   public  void permutation(int[]nums,int begin,List<List<Integer>> result){
           if(begin==nums.length){
               List<Integer> temp=new ArrayList<>();
               for(int n:nums){
                   temp.add(n);
               }
               result.add(temp);
           }
      else{
           for(int i=begin;i<nums.length;i++){
               swap(nums, begin, i);   
               permutation(nums,begin+1,result);//固定上一个位置，向下遍历
               swap(nums, begin, i);//将数组变成原来的样子，循环固定另一个数
           }
       }
   }

   public  void swap(int[] A, int x, int y) {
       int tmp = A[x];
       A[x] = A[y];
       A[y] = tmp;
   }

8.最优数组

给定俩个数组A，B大小相同，使得A[i]>B[i]的个数最多
Input: A = [12,24,8,32], B = [13,25,32,11]
Output: [24,32,8,12]
思路：将A排序，遍历B，每次从A中找出比B[i]当前值大的最小的A中的值，若没有，则将A当没有
     被用过的值的最小值填充进去

public int[] advantageCount(int[] A, int[] B) {
       Arrays.sort(A);
       int[] result=new int[A.length];
       Arrays.fill(result, -1);  //将结果数组都初始化为-1；
       int k=0;
       for(int i=0;i<B.length;i++){
           int temp=B[i];           //以B[i]为标准寻找A中符合的值
           for(int j=0;j<A.length;j++){   
               if(A[j]!=-1&&A[j]>temp){//判断-1是不为了让值重复使用。
                  result[i]=A[j];    
                   A[j]=-1;       //使用过后设为-1，不再使用
                   break;          //找到第一个比B[i]大的值就返回
               }
           }
           if(result[i]==-1){    //判断是否寻找到了适合的值，若没有A将最小值给与
               for(int j=0;j<A.length;j++){
                   if(A[j]!=(-1)){  //同样，找没有赋值过的最小值
                      result[i]=A[j];
                      A[j]=-1;    
                      break;
                   }
               }
           }
       }
       return result;
  }
}
