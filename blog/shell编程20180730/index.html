shell编程

发表于 2018-07-30 | 分类于 linux

所有脚本以#！/bin.bin.sh开头

bash变量

1.变量名在有效的范围内必须唯一，以字母或者下划线打头，中间只能由字母数字下划线组成
2.变量名的长度不能超过255个字符
3.在bash中，变量的默认类型都是字符串类型

定义变量：

变量名=变量值  俩边不能头空格，若变量值有空格需要用双引号引起来
x=5;  或name=“my name"    
x = 5这样的类型是错的，有空格

调用变量：

echo $X   (echo调用变量需要用$符号)

变量叠加：

x=123；
x="$x"456   或者${x}456
echo $x ---123456

设置与删除变量：

set -u ，没有变量会提醒报错
set 查看所有变量
env 查看所有环境变量
unset name   删除name变量

环境变量和用户自定义变量

环境变量全局有用，自定义变量在定义区间有效
在bash中再次bash等于进入一个子区间，只能查看环境变量,pstree查看
export y=1  设置环境变量

位置参数变量

向程序传递调用的参数值

$n  n为数字，$0代表命令本身，$1-$9代表在外输入的第一到第九个参数，十以上参数${10}；
$*  这个变量代表命令行中的所有参数，这里把所有参数看成一个整体
$@    这个变量也代表命令行中多有参数，不过@代表把每个参数区分对待    
$#  命令行的参数的个数
\   反斜杠转意输出原有符号

在shell脚本里调用命令行的参数值
例：./test.sh 35 67
可以在shell脚本使用$1$2来获取35，67对其进行操作

预定义变量

$?  判断最后一次执行命令的返回状态，返回0，正确执行，非0，不正确
$$  当前的进程号pid
$!  后台运行的最后一个进程的进程号

read接受键盘输入

-p：  "提示信息"，在等待read输入时，输出提示信息
      read -p  "please input your name: " -t 30 name 
      //等待用户输入名字作为变量，并且只等候30秒 
-t：  "秒数"，read命令会一直等待用户输入，使用此选项可以指定等待时间
-n：  read命令只会接受指定的字符数，就会执行
       read -p  "please input your sex[M/F]: " -n 1 name
       只接受一个字符就结束
-s：  隐藏输入的数据，适用于机密信息的输入

运算符

declare声明变量类型

declare[+/-][选项]变量名

-：给变量设置类型属性
+：取消变量类型属性
-a:将变量声明为数组
**-i:将变量声明为整数型**
     declare -a movie[2]=tba
     查看数组：echo ${movie[1\*]} 表示查看一个或者所有
-x:将变量声明为环境变量  类似于export，最终执行的declare -x
-r：将变量声明为只读变量  //变量不能操作了，删除也不可以，重启会丢失，临时变量
-p：显示指定变量的被声明的类型

运算方法

使用expr

aa=11
bb=22
sum=$(expr $aa + $ bb)  //这里+俩边要有空格，否则不是数值运算

重要：sum=$(($aa+$bb))        //这里不用在乎空格，好记  推荐
sum=$[$aa+$bb]          //容易和别的命令记混
echo $sum

环境变量配置

source 配置文件 加载配置文件，不然只能注销重新登录才生效

以下配置文件启动就生效

 /etc/profile     //etc下的文件对所有用户生效
/etc/profile.d/*.sh
~/.bash_profile   家目录，每个用户都有，对当前用户生效
~/.bashrc         家目录，定义别名
/etc/bashrc

#正则表达式#
通配符：匹配文件名称
正则：用来匹配文件中的数据，linux中是包含匹配，不是完全匹配

通配符:
-  *  匹配任意内容
-  ？ 匹配任意一个字符
-  [] 匹配括号中的一个内容

字符截取命令

grep "/bin/bash"  /etc/passwd     取出所有可登录的用户
grep "/bin/bash"  /etc/passwd | grep -v "root"  取出所有用户在取反不含root的

cut命令

例：cut -d ":"  -f 1,3 /etc/passwd

-f 截取列
-d 分隔符号： 按照指定字符分割列，默认制表符

grep "/bin/bash"  /etc/passwd | grep -v "root" |cut -f 1，3  -d ":"
取出所有用户在取反不含root的的信息，在将其第1，3，列数据按照"："分割取出

awk命令

awk识别print和printf命令

printf不可以用管道实现

awk  '条件1{动作1}条件2｛动作2｝...'  文件名

sed命令

替换命令

sed [选项] ‘[动作]’  文件名

选项： -n :一般sed会把所有数据输出到屏幕，如果加入此选择只会允许sed命令处理的行输出到屏幕
      -e :允许对输入数据的多条sed命令编辑
      -i ：用sed的修改结果直接修改读取数据的文件，而不是由屏幕输出

动作：
-a：追加，在当前行后添加一行或多行
-c：行替换，用c后面的字符串替换原数据航
-i：插入，在当前行插入一行或多行，d：删除指定行
-p：打印，输出指定行
-s：字符紫环，用一个字符串替换另一个字符串，格式为”行范围s/旧字串/新子串/g”

sort和wc

sort排序：  sort[选项] 文件名
选项：
     -f：忽略大小写
     -n：以数值进行排序，默认使用字符串排序
     -r：反向排序
     -t：指定分隔符
     -k，n[，m]按照指定的字段范围排序，从n到m

wc [选项] 文件名
选项：
-l：只统计行
-w：只统计单词数
-m：只统计字符数

流程控制语句

判断文件是否存在：[-e /root/install.log]    echo $?判断上一条语句执行是否成功

条件判断类型

按照文件类型进行判断

-b 文件    判断该文件是否存在，并且是否为块设备文件（是的为真）
-c 文件    判断该文件是否存在，并且是否为字符设备文件（是的话为真）
-d 文件    判断该文件是否存在，并且是否为目录文件（是的话为真）
-e 文件    判断该文件是否存在（存在的话为真）
-f 文件    判断该文件是否存在，并且是否为普通文件（是的话为真）
-L 文件    判断该文件是否存在，并且是否为符号链接文件（是的话为真）
-p 文件    判断该文件是否存在，并且是否为管道文件（是的话为真）
-s 文件    判断该文件是否存在，并且是否为非空（是的话为真）
-S 文件    判断该文件是否存在，并且是否为套接字文件（是的话为真）

按照文件权限进行判断

-r 文件    判断该文件是否存在，并且是否该文件拥有读权限（有为真）
-w 文件    判断该文件是否存在，并且是否该文件拥有写权限（有为真）
-x 文件    判断该文件是否存在，并且是否该文件拥有执行权限（有为真）
-u 文件    判断该文件是否存在，并且是否该文件拥有SUID权限（有为真）
-g 文件    判断该文件是否存在，并且是否该文件拥有SGID（有为真）
-k 文件    判断该文件是否存在，并且是否该文件拥有Sbit（有为真）

两个文件之间进行比较

文件1 -nt 文件2    判断文件1的修改时间是否比文件2的新（若是则为真）
文件1 -ot 文件2    判断文件1的修改时间是否比文件2的旧（若是则为真）
文件1 -ef 文件2    判断文件1是否和文件2的inode号一致，可以理解为两个文件是否为
同一个文件，这个判断用于判断硬链接是很好的方法

两个整数之间比较

整数1 -eq 整数2    判断整数1是否和整数2相等（相等为真）
整数1 -ne 整数2    判断整数1是否和整数2不相等（不相等为真）
整数1 -gt 整数2    判断整数1是否大于整数2（大于为真）
整数1 -lt 整数2    判断整数1是否小于整数2（小于为真）
整数1 -ge 整数2    判断整数1是否大于等于整数2（大于等于为真）
整数1 -le 整数2    判断整数1是否小于等于整数2（小于等于为真）

字符串的判断

-z 字符串    判断字符串是否为空（为空返回真）
-n 字符串    判断字符串是否为非空（非空返回真）
字串1 == 字串2    判断字符串1是否和字符串2相等（相等返回真）
字串 != 字串2    判断字符串1是否和字符串2不相等（不相等返回真）

多重条件判断

判断1 -a 判断2    逻辑与，判断1和判断2都成立，最终的结果才为真
判断1 -o 判断2    逻辑或，判断1和判断2有一个成立，最终的结果就为真
! 判断    逻辑非，是原始的判断是取反

if…then…fi 语句

[ 条件判断式中间必须有空格 ]

if[ 条件判断式 ]；then    
   程序
fi

或者
if[ 条件判断式 ]
then
程序
fi

if…else if ..else

if [ 条件判断式一 ] ; then

条件一成立时执行操作

elif [ 条件判断式二 ]; then

条件二成立时执行操作

else

当条件1,2均不成立时执行的操作

fi

case…esac语句

case $变量名称 in
"值1"）
    程序段
 ;;  //这里是两个分号代表第一个判断的结束
“值2”）
程序段
;; 
#... 这里可以有多个判断语句的程序段
*)  
    echo ...        //当以上所有的判断均不满足的时候执行的操作。
;;
esac

for循环

 #for循环开始
for var in 1 2 3 4 5 6 7 8
do
   #依次输出列表中的数字
   echo "the number is $var"
done

#for循环开始
for (( i=1;i<5;i++ ))
do
   #输出循环变量i的值
   echo "$i"
done

function函数

function printit(){
 echo -n "Your choice is " #加上-n可以不断行继续在同一行显示
}

read -p "Please input your choice:" choice
case ${choice} in
 "one")
     printit; echo ${choice} | tr 'a-z' 'A-Z' #将参数进行大小写转换
     ;;
 "two")
     printit; echo ${choice}
     ;;
 "three")
     echo "Your choice is three."
     ;;
 *)
     echo "Usage ${0} {one|two|theree}"
     ;;
esac
