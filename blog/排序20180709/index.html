8种排序方法

发表于 2018-07-09 | 分类于 算法

冒泡排序

public void maopao(int[] arr){
    for(int i=0;i<arr.length;i++){
        for(int j=0;j<arr.length-i-1;j++){
            if(arr[j]>arr[j+1]){
                int temp=arr[j];
                arr[j]=arr[j+1];
                arr[j+1]=temp;
            }
        }
    }
}

选择排序

以升序为例，比如在一个长度为N的无序数组中，
在第一趟遍历N个数据，找出其中最小的数值与第一个元素交换，
第二趟遍历剩下的N-1个数据，找出其中最小的数值与第二个元素交换……
第N-1趟遍历剩下的2个数据，找出其中最小的数值与第N-1个元素交换，
至此选择排序完成。

public void selectsort(int[] arr){
    for(int i=0;i<arr.length;i++){
        int min=arr[i]; //每一轮的第一个值作为初始最小值
        int index=i;    //用来记录最小值的位置  
        for(int j=i+1;j<arr.length;j++){
            if(arr[j]<min){
                min=arr[j];  //当前最小值
                index=j;     //记录当前最小值的位置
            }
        }
        if(index!=i){
            arr[index]=arr[i];  //交换位置
            arr[i]=min;
        }
    }
}

插入排序

像扑克摸牌一样，插入即表示将一个新的数据插入到一个有序数组中，并继续保持有序。例如有一个长度为N的无序数组，进行N-1次的插入即能完成排序；第一次，数组第1个数认为是有序的数组，将数组第二个元素插入仅有1个有序的数组中；第二次，数组前两个元素组成有序的数组，将数组第三个元素插入由两个元素构成的有序数组中……第N-1次，数组前N-1个元素组成有序的数组，将数组的第N个元素插入由N-1个元素构成的有序数组中，则完成了整个插入排序。

以下面5个无序的数据为例：
65 27 59 64 58 （文中仅细化了第四次插入过程）
第1次插入: 27 65 59 64 58
第2次插入: 27 59 65 64 58
第3次插入: 27 59 64 65 58
第4次插入: 27 58 59 64 65

public void insertsort(int[] arr){
    for(int i=1;i<arr.length;i++){ //每次当作插入一个新的值，从第二个数开始插入
        int key=arr[i];            //本次循环要插入的值
        int j=i;                   //插入值的位置

        while(j>=0&&arr[j-1]>key){//查找要插入的位置,循环结束时则找到插入位置j 
            arr[j]=arr[j-1];     //移动元素的位置.供要插入元素使用  
            j--;
        }

        arr[j]=key;          //插入指定元素到指定位置  
    }
}

希尔排序

将待排序的数组分成多个子序列，然后对各个子序列进行直接插入排序，待所有排列基本有序后，
最后对所有元素进行一次直接插入排序

希尔排序是按照不同步长对元素进行插入排序，当刚开始元素很无序的时候，步长最大，所以插入排序的元素个数很少，速度很快；当元素基本有序了，步长很小，插入排序对于有序的序列效率很高。所以，希尔排序的时间复杂度会比o(n^2)好一些。由于多次插入排序，我们知道一次插入排序是稳定的，不会改变相同元素的相对顺序，但在不同的插入排序过程中，相同的元素可能在各自的插入排序中移动，最后其稳定性就会被打乱，所以shell排序是不稳定的。

public void shellsort(int[] arr){
    int gap=arr.length/2;   //初始分步设置
    int temp;
    while(gap>0){  //步长最后要成为1进行一下直接插入排序
        for(int i=gap;i<arr.length;i++){ //
            temp=arr[i];  //当前插入值，类似与插入排序，从第二个值开始插入
                            arr[0]是第一个值，arr[gap]是第二个值
            int j=i-gap;  //插入值的前一个值
            while(j>=0&&arr[j]>temp){//当插入值比之前的值小的时候
                arr[j+gap]=arr[j];    //移动元素的位置
                j=j-gap;        //继续循环看之前有没有比插入值小的值
            }
            arr[j+gap]=temp;最后得到的j+gap就是最后要插入的位置
        }
        gap/=2;
    }
}

快速排序

快速排序是C.R.A.Hoare于1962年提出的一种划分交换排序。它采用了一种分治的策略，该方法的基本思想是：

1．先从数列取出一个数作为基准元。//可自行确认
2．分区过程，将比这个数大的数全放到它的右边，小于或等于它的数全放到它的左边。
3．再对左右区间重复第二步，直到各区间只有一个数。

挖坑法：

思路是先将最左边的值保存下来，作为key值。这时候最右边的值被取出去，最右边就相当于有了一个坑，我们从左向右进行遍历，找到一个比key大的数就把它填到这个坑里，这时候坑就相当于之前大数的位置，我们又从右向左进行遍历找比key小的数，找到后再次填到坑里。依次类推

public void test1(){
    int [] arr={3,1,5,2,9,8,4,6,7};
    quicksort(arr,0,arr.length-1);
    System.out.println(Arrays.toString(arr));
}
第一步：key=arr[0]=3作为关键字，此时arr[0]成为了一个坑
第二步：从最右边开始寻找比3小的值，最后寻找到2，角标right=3，将值2赋给arr[0]，
       此时结果为：2，1，5，2，9，8，4，6，7，然后arr[right]=2为新的坑
第三步：从左边left++开始寻找比key大的值，最后寻找到5，将5赋予之前的坑
        arr[right--]=arr[left]即arr[3]=5，然后arr[2]=5为新的坑
        此时结果为2，1，5，5，9，8，4，6，7
第四步：循环后left=right，执行arr[left]=key，将key填入最后的坑中
        2，1，3，5，9，8，4，6，7，第一次划分就完成了

public void quicksort(int[] arr,int low,int high){
    int left=low;
    int right=high;
    if(left>=right) //左边大于等右边时退出循环
        return;
    int key=arr[left];  //首次将坑设置到数组的最左边，也就是用来排序的值
    while(left<right){
        while(left<right&&arr[right]>key) //寻找右边比key小的坐标，大的原地不动
            right--;    //
        if(left<right)
            arr[left++]=arr[right];  //在右边找到比key小的值后，把值添加到‘坑’里
                            这里left坑被填后，需要角标后移，开始下一步寻找
        while(left<right&&arr[left]<key)//然后寻找左边比key大的值
            left++;
        if(left<right)
            arr[right--]=arr[left]; //之前的right已经赋予了坑，所以这是一个新的坑
                                     将找到比key大的值放在这里
    }
    arr[left]=key;    //循环结束后，把key放入坑中，就得到左边比key小，右边比key大了
    quicksort(arr,low,left-1);  继续递归下去
    quicksort(arr,left+1,high);
}

归并排序

public static void main(String[] args) {
     int[] arr={3,1,5,7,85,9,4,6};
      int[] temp=new int[arr.length];//临时数组，用来暂时存储每次归并排序的值
        mergeSort(arr,0,arr.length-1,temp);  
       System.out.println(Arrays.toString(arr));
}

public static void mergeSort(int arr[],int left,int right,int[] temp){
    if(left<right){
        int mid=(left+right)/2;   
        mergeSort(arr,left,mid,temp);  //一直二分
        mergeSort(arr,mid+1,right,temp);
        merge(arr,left,mid,right,temp);//最后从最底层开始归并
    }
}
public static void merge(int[] arr,int left,int mid ,int right,int[] temp){
    int i=left,j=mid+1,k=0;
    while(i<=mid&&j<=right){
        if(arr[i]<=arr[j])
            temp[k++]=arr[i++];
        else
            temp[k++]=arr[j++];
    }
    while(i<=mid){
        temp[k++]=arr[i++];
    }
    while(j<=right){
        temp[k++]=arr[j++];
    }
    k=0;
    while(left<=right)
        arr[left++]=temp[k++];
}
