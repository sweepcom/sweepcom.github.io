linux基本指令

1、linux命令格式

[root@localhost ~]#
root:当前登录用户。在linux当中管理员账号是root。
localhost：主机名。右击我的计算机→属性可查看。
~ 代表当前所在目录（家目录）。超级用户（管理员）的家/root，普通用户的家/home/user1/等。

（#）代表超级用户的提示符。普通用户提示符为$。

2.命令格式
命令 [选项] [参数]
注意：
个别命令使用不遵循此格式；
当有多个选项时，可以写在一起；
简化选项与完整选项：-a等于–all等。

3.查询目录中的内容：Is(list files) 例：ls -l /etc
Is [选项] [文件或目录]
选项：
-a 显示所有文件，包括隐藏文件（以.开头的。隐藏文件多是系统文件，为了不让用户随意修改才隐藏。）
-l 显示详细信息（包括权限等）
-d 查看目录属性
-h 人性化显示文件大小
-i 显示inode（id号，方便查找）

4.权限
-rw-r–r– 第一位（-）为权限，往后每三位为一组，代表文件的u所有者（rw-）、g所属组（r–）和o其他人（r–）。r-4，w-2，x-1
权限特殊类型有七种：文件（-）、目录（d）、软连接文件（l）、块设备文件、字符设备文件、套接字文件和管道文件
基本权限为r读 w写 x执行：所有者有rw权限即读写，所属组有r权限即读权限，其他人有r权限即读权限。

sudo chmod 777 aaa （赋予读写执行权限）

2.目录操作

建立目录 ： mkdir — a

建立多级目录 ： mkdir -p a/b

· 简化操作
cd ~ 进入当前用户的家目录
cd 进入当前用户的家目录
cd - 进入上次目录
cd .. 进入上一级目录
cd . 进入当前目录

· 相对路径 ： 参照当前所在目录，进行查找
如： [root@imoc ~]# cd ../usr/local/src/

· 绝对路径 ：从根目录开始指定，一级一级递归查找。在任何目录下，都能进入指定位置
如： [root@imoc ~]# cd /etc/
->查询所在目录位置 ：pwd

删除目录：

rmdir 只能删除空白目录

touch 文件名——->创建文件rm -r 删除的目录（会有提示）

rm -rf 强制删除文件和目录（remove）

-r 删除目录 （递归 recursive）

-f 强制 （false ，不写会提示）

复制目录

cp（copy）命令
-r 复制目录
-p 连带文件属性复制

-d 若源文件是链接文件，则复制链接属性

-a 复制时间等所有信息，相当于 -pdr

mv 剪切或改名

源文件与目标文件在不同目录：剪切

源文件与目标文件在同一目录：重命名

pwd:显示路径

常用工作目录的作用

/bin /sbin /usr/bin /usr/sbin 都用用于保存系统命令的

/bin /usr/bin 保存所有用户都可操作的命令

/sbin /usr/sbin 保存root可操作的命令

/boot 保存用户启动数据

/dev 特殊文件保存目录

/etc 系统默认配置文件保存目录

/root 超级用户家目录

/home 普通用户家目录

/lib 函数库目录

/misc /media /mnt 设备挂载目录（老Linux系统只有/mnt目录）

/proc /sys 内存挂载点，不能操作

3.搜索命令

文件搜索命令locate（只能文件名搜索）
命令搜索命令whereis与which
文件搜索命令find（全局搜索）：

find [搜索范围]（搜索条件）

find / -name sad（在根目录下按照sad名称搜索）

通配符：需要加引号才能识别

*（匹配任意内容）

？（匹配任意一个字符）

[]（匹配任何一个中括号中的内容）

find . 查看当前目录下所有文件

1、find + 范围（.是当前目录） + -name + 搜索名字

2、find + 范围 + -name + * ：匹配任意内容

3、find + 范围 + -name + ? ：匹配任意一个字符

3、find + 范围 + -name + [] ：匹配括号中的字符,可以多个

4、find + 范围 + -iname + 搜索名字 ：不区分大小写

5、find + 范围 + -user + 用户名字 ：安装所有着搜索

6、find + 范围 + -nouser ：查找没有所以者的文件，通常用于删除垃圾文件

grep字符串搜索命令（在文件当中匹配符合条件的字符串）grep -n “size” sds.cfg（在文件中找到含有size的内容，n代表显示行数）。-V（反选，不包含），-i（忽略大小写）

tail （文件）：从文件尾部开始读

head – ：从文件头部开始读

cat :读取整个文件

wc ：统计行数（wordcount）

more ：分页查看

less ：可控分页

linux各种配置安装

1.jdk安装

1.下载jdk，官网下载或者rz下载到本地上传在解压（rpm -ivh jdk-…）

2.sudo vim /etc/profile(进去配置环境变量)

3.source /etc/profile(加载配置)

4.java -version(查看版本号)

2.tomcat安装

1.下载，解压tar -xzvf tomcat…

2.进入tomcat ，sudo vim conf/servel.xml(在8080，在8443后加URIEncoding=”UTF-8”)

3.在bin目录下，sudo ./startup.sh 启动tomcat

3.mysql安装

1.use mmall;（使用数据库）; source /usr/libei/developer(进入数据库，然后导入表)

2.select * from mmall_product\G;(格式化查询)

3.mysql -u root -p(登录数据库)

4.select user,host,password from mysql.user(查询用户)

4.git安装

1.wget下载

2.安装依赖

sudo yum -y install zlib-devel openssl-devel cpio expat-devel gettext-devel curl-devel perl-ExtUtils-CBuilder perl-ExtUtils- MakeMaker

3.解压后进入git，执行sudo make prefix=/usr/local/git all

​ sudo make prefix=/usr/local/git install，配置环境变量，查看版本号

4.git config –global user.name =”huanglibei”，，

git config –global user.email=”18740450070@163.com“（配置用户名和邮箱）

git config –global core.quotepath off （避免中文乱码）
